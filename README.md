# Admin Dashboard - React + Tailwind CSS

Sistema de administración con autenticación y gestión de usuarios.

## Características

- ✅ Autenticación JWT
- ✅ Gestión de usuarios (CRUD)
- ✅ Dashboard responsivo
- ✅ Tailwind CSS para estilos
- ✅ API REST integration

## Tecnologías

- React 19
- Tailwind CSS
- React Router DOM
- Context API

## Instalación

```bash
npm install
npm start
